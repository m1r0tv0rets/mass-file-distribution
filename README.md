Инструкция по работе:

1. Выберите исходный файл, который хотите распространять.
2. Укажите основную папку, в которой находятся подпапки, куда будет производиться копирование файла.
3. Щёлкните кнопку «Отправить файл» для начала массового копирования.

Особенности:

- Используется модуль shutil для копирования файла.
- Диалоговые окна позволяют удобно выбирать файл и папку.
- Подтверждается успешность завершения операции
